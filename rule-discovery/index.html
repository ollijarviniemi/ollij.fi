This page collects eight different games with 90 rounds in total. Your task is to figure out a hidden regularity. In some games you press the "yes" and "no" buttons and clear a round once you get enough examples right in a row. In other games you work out the rule from examples and then answer test cases correctly, which check whether you've found the right rule.
1. Colorful Shapes
?
You see four shapes, each with a color and a shape. Some sets of shapes follow a secret rule and some don't. Your task is to work out the rule and say, for each set, whether it satisfies the rule or not. You clear the round after answering 10 times correctly in a row.
2. Grids
?
You see a 6×6 grid in which some cells are black. On each round some grids satisfy the rule and some don't. You clear the round after answering 15 times correctly in a row.
3. Points in the Plane
?
You see points arranged on a grid. Again there is a rule that some point arrangements satisfy and some don't. You clear the round after sorting 15 arrangements correctly in a row.
4. Number Sequences
?
You see the first numbers of a sequence. Your task is to figure out the rule that determines how the sequence continues. Give the next number in the sequence until you get two right in a row.
5. The Three-Number Game
?
On each round there is a rule that some number triples satisfy. You are given one example of a triple that satisfies the rule. You can then try other triples and find out whether they satisfy the rule or not. When you're ready, you can write your rule in the text field and move on to the test phase. In the test phase you must classify ten triples correctly. Always aim to pass the test phase on your first try!
6. Function Machines
?
You see a "machine" that takes in one number and returns another number. Your task is to figure out what rule the machine follows. You can test the machine with different numbers. When you're ready, just like in the previous game, you can write the rule in the text field and move on to the test phase. Try to pass the test phase on your first try!
7. Function Machines, Part 2
?
The same as the previous game, but now the machine takes in two numbers and outputs one number. Your task, again, is to work out by experimenting what rule the machine follows.
8. Function Machines, Part 3
?
The same as the previous two games, but now the machine takes in five numbers and outputs one number.
